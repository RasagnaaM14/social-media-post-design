# ☕ Coffee shop Promotional Poster

A modern promotional poster designed to advertise a café's **Iced Cold Coffee** special. Created as a graphic design project to practice poster design, layout, and branding.

## Features
- Clean and minimal design
- Coffee-inspired color palette
- Attractive typography
- 20% discount badge
- Social media icons
- Call-to-action button
